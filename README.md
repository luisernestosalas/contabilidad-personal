# 📊 Módulo de Contabilidad Personal

Una aplicación web sencilla y eficiente para el control de finanzas personales, desarrollada con tecnologías web estándar.

## 🚀 Demo en Vivo

Prueba la aplicación aquí: [Contabilidad Personal](https://contabilidad-personal-3y5kbv0zp-luisernestosalas-2775s-projects.vercel.app)

## ✨ Características

- **📈 Registro de Ingresos y Gastos**: Añade transacciones de forma rápida e intuitiva
- **💰 Balance Total**: Visualiza tu situación financiera actual en tiempo real
- **📅 Filtros por Fecha**: Consulta movimientos por períodos específicos
- **💾 Persistencia Local**: Tus datos se guardan automáticamente en el navegador
- **📱 Diseño Responsivo**: Funciona perfectamente en desktop y móvil

## 🛠️ Stack Tecnológico

- **HTML5**: Estructura semántica y accesible
- **JavaScript (Vanilla)**: Lógica de aplicación sin dependencias externas
